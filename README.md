## Project Setup

- This repository uses Git LFS. Install it [here](https://git-lfs.com/) and afterwards run git lfs pull to pull all assets
- Install Godot v4.5.1 with C# Support.
- Open the project in Godot

## Roadmap

- [ ] Milestone 1
    - Player Movement
    - A small island
    - Grow plants
    - Buy/Sell plants
    - Buy new grow plots and be able to place them somewhere on your island
- [ ] Milestone
    - Day/Night cycle
    - Plants need to be watered before going to sleep and grow one state further after night cycle
